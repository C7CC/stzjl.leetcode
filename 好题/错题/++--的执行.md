## ++/--的执行

### ++/--的运算只能对自变量作用而不能对字面量（值）作用

```java
int i = 1;
i = ++(i++);//此处会报错
```

### i++后会变为一个值而++/--只能对自变量作用而不能对值作用所以报错

```java
int i = 1;
i = ++(i + 1);
```

### 同理此处也会报错

## 细节

```Java
public void test() {
    int a = 10;
    System.out.println(a++ + a--);
}//会输出21					
```

​				 此时a的值		  过后a的值

a++ 相当于    a = a，		     a = a + 1

a--  相当于 	a = a，			 a = a - 1

++a 相当于    a = a + 1，	   a = a

--a 相当于 	 a = a - 1，		a = a

在运算时也生效
